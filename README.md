# java-industrial-smart

欢迎来到《Java × 工业智能》！本项目旨在帮助Java开发者进入工业智能领域，从工业通信协议到边缘计算，用熟悉的Java语言探索工业世界。

## 🎯 专栏特点
- 🛠️ **实战驱动**：每篇文章都有完整可运行的代码示例
- 🚀 **循序渐进**：从基础到应用，构建完整知识体系
- 📚 **贴近实际**：结合实际工业场景，解决真实问题
- 💡 **开源开放**：所有代码开源，欢迎贡献和讨论
